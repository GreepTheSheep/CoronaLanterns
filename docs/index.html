The Guiding Lanterns

(Known as Corona's Lanterns)

Official bot of the Kingdom of Corona

Hello! This is The Guiding Lanterns, a bot created by Greep#3022 and Ajam#3536.

The Guiding Lanterns is a bot that has been interested in Kingdom of Corona, which is a Reddit community of Tangled. It is for this reason that this bot is very interested on the Tangled movie.

List of commands:

  • !quote : Send a random quote from Tangled (and the series also)
  • !screenshot : Take a screenshot from the movie or from the series
  • !wolfram or !full-wolfram : What would you like to know about ?

Commands for supporters:

  • !say : Makes the bot talking !
  • !setgame : Sets a "Playing ..." status
  • !setwatch : same as setgame but is "Watching ..."
  • !setlisten : Same as setgame but is "Listening to ..."

If you have any problems with the bot:

  • !bug : Report any bug to devs
  • !suggest : Suggest an improvement
